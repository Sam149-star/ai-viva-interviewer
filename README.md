# AI Interview Pro 🚀
An AI-powered interview coaching tool that uses Voice-to-Text and LLM analysis to grade user responses.

## 🛠 Tech Stack
- **Frontend:** Streamlit
- **Voice-to-Text:** OpenAI Whisper (via Replicate)
- **LLM Analysis:** Cohere API
- **Data Handling:** Pandas & Python

- **Person A:** UI & Integration Lead
- **Person B:** AI Pipeline Engineer
- **Person C:** Data & Analytics Specialist
